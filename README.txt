Put your hero background video file(s) here, e.g. hero.mp4

Then in the Admin Panel (admin.html), under "Background Video", choose
"Use file from assets folder" and type:  assets/videos/hero.mp4

Tips:
- MP4 (H.264) format works everywhere.
- Keep the file under ~15MB and around 15-30 seconds, looped, no audio needed.
- Compress with Handbrake or a similar tool for fast loading.
